About

Alces Flight Directory provides you with the ability to manage the users and groups available within your Alces Flight environment from one centralized location.

You can create, view, modify, and delete users and groups, as well as add and remove users as members of a group. All changes you make will apply throughout your Alces Flight environment, with users and groups available on all clusters.

Alces Flight Directory also provides you with the ability to 'export' a record of your Flight Directory configuration, to facilitate transferring your setup to a new Alces Flight Directory using the 'import' feature.

All these features are available via the Flight Directory command-line interface, which is available using the access details provided.

Flight Directory Access

Alces Flight Directory is ready and waiting to help you manage the users and groups within your Alces Flight environment.

You should log in to Flight Directory using an SSH client, such as the ssh command at your prompt on Linux or macOS, or with PuTTY on Windows.

You can log in as the flight user, using the hostname or the IP address . You will need the SSH key you provided when Flight Directory was launched.

Eject Flight Directory Support

Alces Flight Directory is configured with the Directory command-line tool for managing the users and groups available within your Alces Flight environment. You are also able to access support for your Flight Directory provided by Alces Flight Ltd.

However, if needed, you can choose to eject Flight Directory support and gain full access to FreeIPA as well as Linux command-line access via SSH to the Flight Directory appliance.

If you wish to eject your Flight Directory support you should contact the Alces Flight Support team to be provided with an eject code. To proceed with ejection you should enter your eject code and submit the form below.

This will opt you out from any obligation for Alces Flight Ltd to provide you with support for Flight Directory.
